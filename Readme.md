## Libraries Used


- ### Streamlit
- Streamlit’s open-source app framework is a breeze to get started with. It’s just a matter of:
```sh
    pip install streamlit
```
- ### To check if its installed
```sh
    streamlit hello
```

- ### Pandas
```sh
    pip install pandas
```
- ### Plotly
```sh
    pip install plotly
```
## Run the script

```sh
    streamlit run Covid-19_EDA.py
```
